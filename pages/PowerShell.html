create_users.ps1 Active Directory
Automatisation de création d'utilisateurs

Ce script permet d'importer une liste d'utilisateurs depuis un fichier CSV et de créer les comptes dans l'AD avec des mots de passe temporaires.

# Exemple de code
                                    $users = Import-Csv "C:\scripts\users.csv"
                                    foreach ($user in $users) {
                                    New-ADUser -Name $user.name -Email $user.mail
                                }
